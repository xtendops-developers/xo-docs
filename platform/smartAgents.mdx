The **SmartAgents** page is your central hub for discovering, creating, and managing SmartAgents.

## Overview

The SmartAgents page provides a comprehensive view of all SmartAgents in your account, including those you've built and pre-configured templates from the library.

## Page Sections

### Hero CTA

The top of the page features a call-to-action hero section that helps you get started quickly.

**"Discover and build SmartAgents powered by XtendOps"**

Create intelligent agents that automate conversations and enhance customer experiences.

**Action:** Click the **Create SmartAgent** button to launch the [SmartAgent Builder](/platform/smartAgentBuilder) and start building a new workflow from scratch.

### My SmartAgents

View and manage all SmartAgents you've created for your account.

**Features:**
- **SmartAgent Cards** - Visual preview of each SmartAgent showing its canvas/workflow
- **SmartAgent Type Badge** - Displays the type (Automation, Copilot, etc.)
- **Usage Metrics** - See message volume per day for each SmartAgent
- **Owner Information** - View who created the SmartAgent
- **Quick Actions:**
  - **Demo** - Test the SmartAgent in a live conversation
  - **View Profile** - Navigate to the [SmartAgent Profile](/platform/smartAgentProfile) page to see details and access the builder

### SmartAgent Library

Browse ready-to-use SmartAgent templates with pre-defined workflows and configurations to help you get started quickly.

**Features:**
- **Template Cards** - Pre-configured SmartAgents for common use cases
- **Category Filters** - Browse by automation type, industry, or use case
- **Type Filters** - Filter by SmartAgent type (Automation, Copilot, etc.)
- **Key Features Tags** - See what capabilities each template includes
- **Quick Actions:**
  - **Demo** - Test the template SmartAgent before cloning
  - **Clone** - Copy the template to your account for customization

**Use Cases:**
The library includes templates for common scenarios like customer support automation, order tracking, returns processing, FAQ handling, and more.

## Navigation

From this page, you can:
- Create new SmartAgents from scratch
- Access and edit existing SmartAgents via their profile pages
- Demo SmartAgents to test functionality
- Clone library templates to quickly implement proven workflows
