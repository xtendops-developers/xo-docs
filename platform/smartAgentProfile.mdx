The **SmartAgent Profile** page provides an overview of a specific SmartAgent's configuration, performance metrics, and recent activity.

<div align="center">![SmartAgent Profile](/images/platform/profile.png)</div>

## Overview

The profile page displays key information about the SmartAgent at a glance, including its status, type, description, and features.

**SmartAgent Details:**

- **Status** - Active or inactive indicator
- **Type** - SmartAgent type (Copilot, Automation, etc.)
- **Category** - Use case category
- **Description** - Purpose and functionality summary
- **Key Features** - Highlighted capabilities

## Agent Metrics

Performance metrics tracked over time for the SmartAgent.

**Available Metrics:**

- **Avg. Feedback Score** - Average quality rating with comparison to last month
- **Conversation Count** - Total number of conversations handled
- **Messages per Day** - Daily message volume

## Activity Overview

Chart showing AI message volume generated over a selected time period with comparison to previous periods.

**Features:**

- Date range selector (Last 7 days, 30 days, etc.)
- Time-series visualization
- Period-over-period comparison

## Latest Conversations

Table displaying recent conversations with the SmartAgent, including conversation IDs, creation dates, and message counts.

## Latest Activity

Activity log showing recent updates and changes to the SmartAgent, including profile updates, step modifications, and version deployments.

## Integrations

Connection information for external systems, including:

- **Chatbot URL** - API endpoint for integrating with external platforms
- **Sample POST Request** - Example API call format
